# Онлайн-платформа обучения

Этот проект представляет собой онлайн-обучающую платформу на Django с поддержкой курсов, уроков, подписок и платежей.

## Запуск проекта через Docker

### 1. Подготовка окружения
Скопируйте шаблон `.env`: cp .env.example .env
Отредактируйте файл `.env`, указав свои значения (например, `SECRET_KEY`, пароли и т.д.).

### 2. Запуск сервисов
Запустите все сервисы одной командой:docker-compose up --build
> Первый запуск может занять больше времени из-за сборки образов и установки зависимостей.

### 3. Проверка работоспособности
- **Бэкенд (Django)**:  
  Откройте [http://localhost:8000](http://localhost:8000) — должна открыться страница сервера (или ошибка 404, если нет главной страницы — это нормально, API работает).
- **База данных (PostgreSQL)**:  
  Проверяется автоматически при старте `backend` и `celery`. Ошибки подключения будут видны в логах.
- **Redis**:  
  Проверьте, что контейнер `olp_redis` запущен: docker logs olp_redis
- **Celery Worker**:  
  В логах должен появиться старт воркера:  docker logs olp_celery
  - **Celery Beat**:  
  Должен показывать расписание периодических задач: docker logs olp_celery_beat

### Остановка проекта
bash docker-compose down

Для полной очистки (включая данные БД): docker-compose down -v
